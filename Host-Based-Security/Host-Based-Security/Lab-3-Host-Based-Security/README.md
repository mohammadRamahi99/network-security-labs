## Objective

The objective of this lab was to configure and secure Windows hosts in an Active Directory environment using Group Policy, PowerShell, and domain-based access controls. The lab focused on enforcing host-based security controls, hardening endpoints, and centrally managing authentication and authorization across domain-joined systems.

This lab demonstrates enterprise-level host security administration, policy enforcement, and identity-based access control.

- Windows Server (Primary Domain Controller)

- Active Directory Domain Services (AD DS)
- Group Policy Management Console (GPMC)
- PowerShell
- SMB File Services
