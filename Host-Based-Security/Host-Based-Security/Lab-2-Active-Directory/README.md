## Objective
The objective of this lab was to deploy and secure a Windows Active Directory environment.
This included creating a new domain, designing Organizational Units (OUs), managing users and groups,
adding a secondary domain controller using PowerShell, and enforcing security controls through
Fine-Grained Password Policies.

This lab demonstrates enterprise-level identity management, authentication, authorization,
and domain security administration.
